# 📧 Email Spam vs Not Spam Detector

Email Spam vs Not Spam Detector is a machine learning–based web application that classifies email messages as **Spam** or **Not Spam** using Natural Language Processing (NLP). The application analyzes the email content, applies TF-IDF vectorization, and uses a trained ML model to provide real-time predictions through an interactive Streamlit interface.

**Key Features**
- Email text spam classification
- NLP preprocessing with TF-IDF
- Machine learning–based prediction
- Interactive and colorful Streamlit UI
- Deployed on Streamlit Cloud

**Technologies Used**
Python, Machine Learning, NLP, Scikit-learn, Streamlit, GitHub

**How It Works**
1. User enters the email content.
2. Text is transformed using TF-IDF vectorization.
3. The trained model predicts spam or not spam.
4. Result is displayed with visual indicators.
